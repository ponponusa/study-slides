## 構成（約30分）

1. **導入：自由 vs 制約** (5分)
   - 「自由に作れる」は本当に良いことか？
   - 道路と鉄道の違いから考える

2. **軌間の話** (10分)
   - 新幹線と在来線はなぜ直通できないのか
   - 軌間（規格）の違いがもたらす分断
   - 規格統一の恩恵とベンダーロックイン回避

3. **レールという制約の恩恵** (10分)
   - 制約があるから速く走れる
   - REST API、SQL、HTTP... 規格があるから世界が繋がる

4. **落とし穴** (5分)
   - レールを間違えると敷き直しが大変
   - 技術選定の責任は重い

## Key Takeaways

1. **レールがあるから速く走れる** — 制約は敵ではなく味方
2. **規格に従えばベンダーを変えても走れる** — ロックイン回避
3. **レールは敷き直せない** — 技術選定は慎重に
