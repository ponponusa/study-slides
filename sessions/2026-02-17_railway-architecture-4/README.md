## 構成（約30分）

1. **乗客が10倍になったらどうする？** (5分)
   - スケールアップ：車両を大きくする（限界がある）
   - スケールアウト：本数を増やす/複線化する（制御が難しい）

2. **破壊的イノベーション（新幹線）** (10分)
   - 在来線のレールでは300km/hは出せない
   - 「別線」を敷いた＝リプレイス/リアーキテクチャ
   - しかし、莫大なコストと期間がかかる
   - 並行稼働の地獄：古いシステムを動かしながら新しいシステムを構築

3. **トポロジー（路線図）の設計** (10分)
   - ハブ＆スポーク（中央集権） vs 直通運転（分散型）
   - どこに「ハブ」を置くかで流れが決まる

4. **シリーズ全体のまとめ** (5分)
   - 技術（車両）は変わる
   - でも、トポロジー（路線図）とルール（レール）を設計する視点は一生使える

## Key Takeaways

1. **限界を超えるには別線が必要** — リアーキテクチャの判断
2. **並行稼働は地獄** — リプレイスの現実
3. **トポロジー設計は一生モノ** — 技術は変わっても設計力は残る
