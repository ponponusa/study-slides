# ☕ Intermission

## テーマ

荒野の歩き方 — 理想の地図と、現実の獣道

## 位置づけ

Phase 1（鉄道から学ぶアーキテクチャ）と Phase 2（AI活用）の間を埋める「ワクチン」セッション。

正式なナンバリングではなく、インターミッション（幕間）として位置づける。

## 狙い

- 「理想を知った状態で、強力な武器（AI）を持つと、現実を破壊したくなる」リスクを防ぐ
- 技術スタック（道具）とアーキテクチャ（設計）の違いを理解する
- レガシーコードへのリスペクトを持つ
- AIを「優秀な新卒」として適切に活用する視点を養う

## 構成（約40分）

1. **路線図と実際の線路** (5分)
   - Phase 1 は「綺麗な路線図」の話だった
   - 現場は「勾配あり、カーブあり、老朽化あり」の線路

2. **車両と路線設計は別物** (10分)
   - 新型車両（技術スタック）を入れても、路線設計（アーキテクチャ）がダメなら遅い
   - 「道具のせいにするな」

3. **廃線にできない在来線** (10分)
   - レガシーコード = ビジネスの歴史
   - チェスタトンの柵：理由を調べてから判断せよ

4. **自動運転と運転士** (10分)
   - AI = 自動運転システム（教科書通りに走る）
   - 人間 = 運転士（現場の文脈を知っている）
   - 両者の協働が最強

5. **クロージング** (5分)
   - Phase 2 への橋渡し
   - 「道具に頼るな、知恵を絞れ。そしてAIと共に進もう」
